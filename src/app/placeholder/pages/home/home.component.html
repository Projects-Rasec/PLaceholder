Que es JsonPlaceholder

Es una herramienta para simular una API REST, proporcionando datos ficticios para pruebas y desarrollo.

Mas info

¿Cuando usar?

JSONPlaceholder es una API REST en línea gratuita que puedes usar siempre que necesites datos falsos . Puede estar en un README en GitHub, para una demostración en CodeSandbox, en ejemplos de código en Stack Overflow o simplemente para probar cosas localmente.

Recursos

JSONPlaceholder viene con un conjunto de 6 recursos comunes:

/posts

100 publicaciones

/comments

500 comentarios

/albums

100 álbumes

/photos

5000 fotos

/todos

200 tareas

/users

10 usuarios
